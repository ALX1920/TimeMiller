# 📘 TimeMiller — Overview

## 🛰️ 1. Overview

I created **TimeMiller**, a web application inspired by the time‑dilation concept from _Interstellar_.  
The goal of the project is to simulate:

- Time elapsed on **Earth**
- Equivalent time on **Miller’s planet**
- A dynamic visit counter
- A cinematic UI with animated text colors
- A clear explanation of the time‑dilation rule

The system updates in real time and uses the movie’s release date as the starting point.
